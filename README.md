# Odoo Customer Statement

Installable Odoo 18 accounting addon for generating customer transaction statements with running balances and Excel export.

## Features

- Partner and date-range filters
- Posted receivable/payable entries
- Debit, credit, and running balance
- On-screen preview
- Excel export
- Accounting-user access controls

Module: `portfolio_customer_statement`  
Dependency: `account`  
Python dependency: `xlsxwriter`
